使用 Nushell 获取好像挺跨平台的 shell 体验

引言

Nushell 是一个跨平台的 shell 至少在 Windows 和 Linux (比如我用的 Arch Linux) 上都能用

换这个主要是能在 Windows 上不用 PowerShell
因为 PowerShell 的行为和 zsh 的差太多了 而且界面也差

比如

  • PowerShell 内建了一些 unix 命令 但行为又不一致 得加.exe才能执行自己安装的正常的
  • PowerShell 的Up和Down浏览的历史输入光标保持在前位置而不是末尾
  • PowerShell 的Tab补全不好用 (当然 zsh nushell 也没多好用)

安装

需要安装的工具

  • Nushell
  • scoop (Windows)
    • 其实只有 winget 也行 但我不喜欢用 winget 管非命令行工具
  • starship (主题)
  • carapace-bin (补全)

Windows

用 scoop

Arch Linux

用 paru (或者其他 AUR 工具)

配置

`env.nu`
`config.nu`
`starship.toml`

这里把starship.toml在 Linux 和 Windows 之间共享 方便修改

当然另外两个文件也可以共享但是我懒因为还是有一些特定于平台的配置

效果

已知问题

  • 补全习惯改变 不能通过Space\等连续补全 只能每步Enter确认
  • 补全习惯改变 可以不完整多级路径补全
